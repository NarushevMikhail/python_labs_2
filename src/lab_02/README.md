# Лабораторная работа № 2.
## Цель работы
* Научиться работать с **коллекциями объектов**.
* Понять разницу между **моделью сущности и контейнером объектов**.
* Реализовать **собственный контейнерный класс**.
* Освоить **итерацию по объектам**.
* Реализовать базовые операции управления коллекцией.

## Процесс создания коллекции: 
 В рамках данной лаборатной работы мною был создан класс-контейнер UserList, предназначенный для хранения и управления объектами класса User из первой лабораторной работы. Коллекция обеспечивает удобный интерфейс для работы с группой пользователей, включая добавление, удаление, поиск, сортировку и фильтрацию элементов.

## Методы и их функционал для работы с коллекцией.
* ```add()``` – добавляет пользователя в коллекцию после проверки типа и уникальности (нет дубликата объекта, nickname и login).
  
* ```remove()``` – удаляет указанного пользователя из коллекции.
  
* ```remove_at()``` – удаляет и возвращает пользователя по указанному индексу.
  
* ```get_all()``` – возвращает внутренний список всех пользователей.
  
* ```find_by_nickname()``` – ищет и возвращает список пользователей с указанным nickname.
  
* ```find_by_login()``` – ищет и возвращает список пользователей с указанным login.
  
* ```find_by_role()``` – ищет и возвращает список пользователей с указанной ролью.
  
* ```sort_by_nickname()``` – сортирует коллекцию по nickname (по возрастанию или убыванию).
  
* ```sort_by_login()``` – сортирует коллекцию по login.
  
* ```sort_by_role()``` – сортирует коллекцию по роли.
  
* ```get_with_profile()``` – возвращает новую коллекцию пользователей с заполненным профилем (bio, возраст или город).
  
* ```get_by_role()``` – возвращает новую коллекцию пользователей с указанной ролью.
  
* ```get_admins()``` – возвращает новую коллекцию администраторов.
  
* ```get_users()``` – возвращает новую коллекцию обычных пользователей.
  
* ```get_moderators()``` – возвращает новую коллекцию модераторов.

## Магические методы. 
* ```__len__()``` – возвращает количество пользователей в коллекции (для функции len()).

* ```__iter__()``` – делает коллекцию итерируемой (для цикла for).

* ```__getitem__()``` – позволяет получать пользователя по индексу через квадратные скобки.

* ```__str__()``` – возвращает строковое представление коллекции с количеством и списком пользователей.
